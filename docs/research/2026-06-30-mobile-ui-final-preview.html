可开工切片

这个预览可直接拆成三刀，先做信息架构和 Today Home，再做 Records/detail，最后做 Insight/Settings/Packs 分离。

Implementation-ready
Slice 1：导航 + Today Home

底部中心记录 action、Home header、今日回顾卡、最近记录、继续记录 CTA。 统一所有记录入口打开同一个 Compose Sheet。

Slice 2：Records + Detail

Timeline 用户文案收敛为 Records，按天分组、过滤 chips、source-first detail、 派生 section 默认展开/折叠规则。

Slice 3：Insights + Boundary

洞察页三块内容、追问进入 Chat 并携带来源，Packs 只做 Pack 管理， Settings 接管权限、模型、备份和 Trace。

必须测

Home CTA 和底部记录按钮打开同一个 Sheet；保存后回到 Home；Recent All 进入 Records。

必须测

详情页 Source 默认展开；Summary/Todo 展开；Memory/Insight 折叠；来源行可追溯。

必须测

洞察问题进入 Chat 并带上下文；空洞察显示轻量空状态和去记录入口。

必须测

Packs 与 Settings 入口不再重复混乱；旧路由兼容或清晰跳转。

最终状态

可以开工：推荐先做 Slice 1。仍需你最后确认 3 件事： 中心记录按钮是否占底部槽位；Insights 是全页还是轻量 sheet； Settings 是否第一刀就从 Packs 完全拆出。
