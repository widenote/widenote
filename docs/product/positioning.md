# Product Positioning

WideNote / 广记 is a mobile-first, local-first personal record, memory, and agent runtime.

For everyday users, the product should feel like:

> A private record app that helps organize life fragments.

For advanced users and the community, it should become:

> A programmable personal context runtime with installable Agent Packs.

## First Users

The first audience is not everyone who writes notes. It is:

- High-frequency recorders
- AI-heavy users
- Creators, developers, researchers, founders, and independent workers
- People who want self-organization, memory, and context reuse
- Users willing to try plugins, Agent Packs, and custom workflows

## Default Loop

The default loop is:

```text
quick capture -> timeline / cards -> memory -> insight
```

The default product should not force todos, document generation, exports, or graphical agent flows. Those should be optional Agent Packs.

## Boundaries

WideNote is not:

- A pure AI diary
- A pure PKM editor
- A pure todo app
- A generic agent platform
- A default full-surveillance lifelog

WideNote is a local-first personal context system. Records are the entry point, Agent Packs are the processing mechanism, and memory, insight, action, and export are possible outcomes.
