# WideNote Agent Assets

This directory contains repository-scoped agent assets. `AGENTS.md` remains the
canonical instruction file; assets here should stay small and task-specific.

- `skills/`: Codex skills for WideNote code-change orchestration.

When adding a durable skill, update `.agents/skills/README.md` and
`docs/agent-context/project-map.md` in the same change.
